# Smart-Lender
🏦 Smart Lender – Loan Eligibility Prediction System
## 📌 Overview
Smart Lender is a Machine Learning-based web application that predicts whether a loan application should be approved or rejected based on applicant details.

## 🚀 Features
- Loan eligibility prediction
- Multiple ML algorithms
- Flask web interface
- Real-time predictions
- User-friendly UI
- Loan rejection suggestions

## 🛠 Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Scikit-learn
- Pandas
- NumPy

## 📂 Project Structure

Smart-Lender/
├── app.py
├── model.pkl
├── requirements.txt
├── README.md
├── templates/
├── static/
└── dataset/


## ▶️ Installation

```bash
git clone <repository-url>
cd Smart-Lender
pip install -r requirements.txt
python app.py
```

## 🌐 Usage

1. Open the application.
2. Enter applicant details.
3. Click **Predict**.
4. View the loan approval/rejection result.
